# Bundled basis-set data

These JSON files are Gaussian basis-set definitions retrieved from the
[Basis Set Exchange](https://www.basissetexchange.org/) (elements H–Ar), in the
native BSE JSON format. They are redistributed here under the **CC-BY 4.0**
license.

If you use the Basis Set Exchange data, please cite:

> B. P. Pritchard, D. Altarawy, B. Didier, T. D. Gibson, T. L. Windus,
> "A New Basis Set Exchange: An Open, Up-to-Date Resource for the Molecular
> Sciences Community", *J. Chem. Inf. Model.* **2019**, 59(11), 4814–4820.
> [doi:10.1021/acs.jcim.9b00725](https://doi.org/10.1021/acs.jcim.9b00725)

Each basis set additionally has its own primary literature references (carried
in the JSON `references` field); please cite those when using the corresponding
basis set.

Bundled sets: `sto-3g`, `6-31g`, `def2-svp`, `cc-pvdz`. On-demand download of
other sets/elements (with local caching) is planned — see `DataPlan.md`.
